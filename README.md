# WeddingInvitation
Wedding Invitation Template
All content is driven from a single `CONFIG` object in `/lib/wedding-config.ts` — just edit that file to customize names, dates, venue, and links. The design uses a cream/blush-pink/champagne-gold palette with Playfair Display and Great Vibes fonts, floating petal animations, scroll-reveal effects via IntersectionObserver, and a subtle paper-texture overlay.
